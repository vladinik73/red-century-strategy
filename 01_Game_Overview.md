# Red Age -- Game Overview

## Vision

Near-future geopolitical turn-based strategy (PvE first).

## Core Pillars

-   Minimalistic UI (Polytopia-inspired)
-   Realistic but simplified map
-   City-centric territory control
-   Three victory paths: Military, Economic, Technological
-   Global technology system
-   One unified transport network per civilization

## Session Length

1--3 hours (PvE)
