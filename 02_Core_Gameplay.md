# Core Gameplay

## Turn Structure

-   Global Action Points (AP/ОД)
-   Sequential turns
-   No undo of unit movement or attack

## Cities

-   50--100 cities (PvE)
-   Integration time: 4 turns
-   Only integrated cities count toward victory

## Transport Network

-   One network connected to capital
-   Recalculated on build/destroy/capture
-   Disconnected nodes excluded from bonus

## Resources

-   Resource tile: 10 units
-   Extract 2 per turn
-   After depletion → low productivity (1 per turn)
