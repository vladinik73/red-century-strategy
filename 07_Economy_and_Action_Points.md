# Economy & Action Points

## AP Formula

Base 5 + floor(total city levels / 5) + floor(technologies / 3) +
stability modifier + network modifier + alliance modifier - occupation
penalty - cyber penalty

## Stability Effects

80--100 → +1 AP 20--39 → -1 AP 0--19 → -2 AP

0 stability → 10% city revolt (becomes neutral)
