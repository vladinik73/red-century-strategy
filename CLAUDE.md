## Проект
**Red Age** — пошаговая web-first стратегия (PvE-first), near-future (2035–2050), вдохновлена Civ + Polytopia.  
Спецификация лежит в `docs/` (разбита на части), **Source of Truth**: `docs/00_meta/SOURCE_OF_TRUTH.md`.

## Золотые правила
1) **Не менять геймдизайн** без явного запроса владельца проекта.  
2) Любое изменение механик → обнови:
   - соответствующий раздел в `docs/**`
   - `docs/00_meta/CHANGELOG.md` (кратко: что/почему)
3) Не дублировать правила. Если правило конфликтует — править **Source of Truth** и синхронизировать остальные разделы.
4) Для новых сущностей и API — поддерживать `schemas/**` (JSON-черновики).
5) При генерации кода — придерживаться web-first UX и минимализма.

## Контекст, который всегда нужно учитывать
- Нет границ стран: есть только границы городов.
- Города предгенерируются, новых не строим.
- Интеграция захвата: 4 хода, только интегрированные города дают бонусы/участвуют в победах/магистрали.
- Магистраль: одна сеть, связанная со столицей.
- Юниты: мгновенное перемещение, без undo, HP — цифра (красная при ≤30% max).
- Атака: кликом по врагу (без кнопок Attack/Move).
- Победы и приоритеты — как в `docs/08_diplomacy/Victory_Rules.md`.
- Кибер/гиперзвук ограничения (столица защищена) — как в `docs/07_units/Advanced_Units.md`.
