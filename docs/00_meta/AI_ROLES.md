# AI Roles — Who Does What

**Дата:** 2026-03-04

---

## Роли

| Роль | Инструмент | Ответственность |
|------|------------|-----------------|
| **System architect + spec guardian** | ChatGPT | Архитектура, целостность спецификации, генерация команд для Cursor/Claude |
| **Auditing + alternative drafts** | Claude | Проверка согласованности спецификации, альтернативные черновики |
| **Execution** | Cursor | Применение патчей, создание файлов, тесты, код |
| **Decisions + acceptance** | User | Финальные решения, приёмка изменений |

---

## Workflow

1. **User** формулирует задачу.
2. **ChatGPT** (опционально) уточняет архитектуру, генерирует детальные инструкции.
3. **Cursor** выполняет: создаёт/редактирует файлы, запускает тесты.
4. **Claude** (опционально) проверяет спецификацию на противоречия.
5. **User** проверяет результат и принимает или отклоняет.
