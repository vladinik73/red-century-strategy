# Как работать с Claude Code в этом проекте

## Рекомендуемый старт
- Попроси Claude прочитать `CLAUDE.md` и `.claude/project.md`.
- Затем используй шаблоны из `.claude/commands/`.

## Правило правок
Если меняется механика — фиксируй в `docs/` и `docs/00_meta/CHANGELOG.md`.

## Примеры
- «Сделай каркас UI layout по docs/10_uiux/PvE_Web_Layout.md»
- «Реализуй формулу ОД из docs/04_economy/Action_Points.md»
- «Добавь систему видимости по docs/03_map/Visibility.md»
