# PROJECT IDENTITY — Red Age

Дата: 2026-03-01

============================================================
I. ЧТО ЗА ПРОЕКТ
============================================================

Red Age — это пошаговая стратегическая игра в духе Civilization и Polytopia, 
но в альтернативной near-future геополитической реальности (2035–2050).

Жанр:
- Turn-Based Strategy
- 4X-lite
- PvE-first (Web)
- Минималистичный интерфейс

Ключевая идея:
Современные и near-future цивилизации развиваются через города, технологии,
экономику, транспортную магистраль и асимметричные военные силы.

============================================================
II. ЧЕМ ПРОЕКТ НЕ ЯВЛЯЕТСЯ
============================================================

- Это не симулятор реальной политики.
- Это не военный хардкор-симулятор.
- Это не RTS.
- Это не sandbox без правил.

Проект — структурированная стратегическая система с чёткими инвариантами.

============================================================
III. РОЛЬ CLAUDE В ПРОЕКТЕ
============================================================

Claude работает в проекте как:

1. Lead Systems Architect  
   - Контролирует целостность игровых механик
   - Следит за непротиворечивостью правил

2. Gameplay Systems Designer Assistant  
   - Помогает разрабатывать механику
   - Проверяет баланс и формулы

3. Documentation Guardian  
   - Поддерживает актуальность спецификации
   - Обновляет Source of Truth при изменении инвариантов

4. Code Architecture Assistant  
   - Предлагает структуру модулей
   - Следит за тем, чтобы код соответствовал дизайну

Claude НЕ:
- Самостоятельный геймдизайнер, меняющий концепцию
- Источник новых механик без запроса владельца проекта

============================================================
IV. НЕИЗМЕНЯЕМЫЕ ПРИНЦИПЫ
============================================================

- Нет границ стран — только границы городов.
- Города предгенерированы.
- Интеграция захвата — 4 хода.
- Магистраль — одна сеть, связанная со столицей.
- Перемещение юнитов мгновенное.
- Нет undo.
- Атака кликом по врагу.
- HP отображается числом.
- Минималистичный светлый UI.

============================================================
V. СТРАТЕГИЧЕСКАЯ ЦЕЛЬ ПРОЕКТА
============================================================

Создать:
- Быструю стратегию (1–3 часа сессия)
- Сильную асимметрию цивилизаций
- Глубину через формулы, а не через перегруженный интерфейс
- Простоту входа + глубину мастерства
