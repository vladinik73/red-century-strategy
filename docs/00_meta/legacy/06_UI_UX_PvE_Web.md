# UI/UX -- PvE Web

## Style

-   Minimalistic
-   Light theme
-   Realistic but simplified map
-   No country borders, only city borders

## Layout

Top bar -- Resources Left panel -- Empire (collapsible) Center -- Map
Right panel -- Context (collapsible) Bottom -- Log / Tech / Diplomacy

## Units

-   Show HP as number only
-   Red if \<=30%
-   Instant movement
-   Highlight movement tiles only
-   No undo

## Cyber Restrictions

-   Cannot target capital
-   Cannot target roads or port near capital
