# MVP Player Journey (v4.17)

Этот документ фиксирует минимальную цепочку экранов и сценарий игрока для MVP (web-first), без режима наблюдателя.

## 1) Цепочка экранов (MVP)

1. **Home**
   - New Game
   - Continue (Autosave)
   - Settings

2. **New Game Setup**
   - Выбор цивилизации игрока: 1 из 7 (Hidden Civilization недоступна игроку в выборе)
   - Сложность AI: `Easy / Normal / Hard / God`
   - Опции матча (минимум):
     - Hidden Civilization: ON/OFF
     - Seed: Auto (показывать необязательно)

3. **Loading / Briefing**
   - Генерация карты 80×80 по `docs/03_map/Map_Generation.md`
   - Коротко: условия победы + стартовые условия

4. **Main Game**
   - Карта + HUD ресурсов + панели юнита/города + End Turn

5. **Victory / Defeat**
   - Тип победы и причина
   - Кнопки: Rematch / Exit

## 2) Автосейв и Continue

### 2.1 Autosave policy (MVP)
- **Автосейв производится в конце хода каждой цивилизации** (см. `docs/01_overview/Turn_Pipeline.md` → PHASE 4).
- В меню **Continue** загружается последний autosave.

Примечание: формат состояния матча определяется `schemas/match.schema.json` (канонический контейнер состояния).

## 3) Порядок ходов (turn order)

### 3.1 Canon (v4.17)
- Порядок цивилизаций **фиксированный** (не случайный).
- **Сначала ходят все ИИ**, игрок ходит **последним**.
- Проверка побед и обновление победных таймеров выполняются **в конце каждого хода цивилизации** (см. Turn Pipeline).

## 4) Сложности AI (MVP)

### 4.1 Presets
Сложности представлены четырьмя пресетами:
- `Easy`
- `Normal`
- `Hard`
- `God`

На уровне MVP допускается реализация сложностей как наборов числовых модификаторов (например, бонус к доходу/науке/ОД или штрафы игроку).
Конкретные числа могут быть определены в отдельном патче баланса.

## 5) Цель по длительности матча

- Целевой ориентир для баланса MVP: **60–90 ходов** до типовой победы.

## 6) End Turn (UX rule)

- Даже если у игрока 0 ОД и нет доступных действий, ход завершается **только по нажатию кнопки End Turn** (без авто-завершения).

## 7) Out of scope (MVP)
- Observer / spectator mode — **не входит** в MVP.
