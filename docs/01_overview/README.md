# Обзор игры

**Red Age** — пошаговая 4X-стратегия (PvE-first), вдохновлённая Civilization + Polytopia, но в мире альтернативной современности/near-future (≈2035–2050).

## Цели
- Простая, быстрая в освоении стратегия (без перегрузки микроменеджментом)
- Сильная вариативность путей победы
- Асимметрия цивилизаций через уникальные юниты и уникальные технологии
- Красивый, читаемый и минималистичный web UI

## Режимы (на текущий MVP)
- PvE: 1 игрок + 6 ИИ (один уровень сложности на всю партию)
- Наблюдение/реплеи (после поражения — переключение перспективы)

> PvP будет добавлен позже (в документе остаются заделы: монетизация, таймеры).
