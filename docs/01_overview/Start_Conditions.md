# Start Conditions (v4.11)

Канонический документ стартовых условий партии (PvE / Web-first).
Цель v1: быстрый, понятный старт в духе Polytopia.

## 1) Стартовые ресурсы игрока
- Деньги (Money): **100**
- Наука (Science): **0**
- Стабильность (Stability): **70** (см. `docs/04_economy/Stability_and_Morale.md`)
- Мораль: вычисляется из стабильности (см. `docs/04_economy/Stability_and_Morale.md`)
- Очки действий (ОД/AP): вычисляются по формуле (см. `docs/04_economy/Action_Points.md`), с инвариантом **min AP = 1**.

## 2) Стартовые технологии
- На старте **нет** открытых технологий (`T0`).
- Производство базовых юнитов разрешено правилами города/производства (см. `docs/02_cities/City_Levels.md`, `docs/07_units/Production_Rules.md`).

## 3) Стартовая столица
- Столица = первый город цивилизации (см. `docs/02_cities/City_Capture.md`).
- Уровень: **City Level = 1**
- Интеграция: столица считается **интегрированной** с начала партии (без 4-х ходов ожидания).
- Защита города по умолчанию: **Defense Level = 0** (`D0`) — защита включается только через улучшение за деньги (см. `docs/04_economy/City_Defense.md`).

## 4) Стартовые юниты
- Игрок начинает с **1** стартового юнита типа **A2 (Разведка)**.
- Параметры стартового юнита берутся из таблицы базовых юнитов (`docs/07_units/Base_Units.md`).
  Если таблица в будущем будет содержать несколько развед-юнитов, стартовый — тот, который явно помечен как `StarterScout`.

## 5) Стартовая видимость
- На старте открыта:
  - вся территория стартовой столицы (видимость города)
  - область видимости стартового юнита (см. `docs/03_map/Visibility.md`)
- Видимость перманентная (2-state UNEXPLORED/VISIBLE), см. `docs/03_map/Visibility.md` (v4.9).

## 6) Нейтральные города и стартовый контакт
- Нейтральные города существуют согласно `docs/03_map/Map_Generation.md`.
- Стартовые условия не гарантируют раннего контакта, но дизайн-цель: первые столкновения до ~20 хода (баланс/генерация).
