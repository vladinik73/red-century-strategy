# Уровни городов

## Общие правила
- Максимальный уровень города: **5**
- Апгрейд уровня стоит **деньги** (не ОД)
- Город должен быть **интегрирован** для повышения уровня
- Стартовый уровень города: 1

## Стоимость апгрейда

| Переход | Стоимость (деньги) |
|---------|-------------------|
| 1 → 2   | 100               |
| 2 → 3   | 200               |
| 3 → 4   | 350               |
| 4 → 5   | 550               |

> Стоимость может корректироваться по результатам балансировки.

## Влияние уровня города

| Параметр | Формула |
|----------|---------|
| Лимит юнитов | = уровень города |
| Авиация | = floor(уровень / 2) |
| Лимит кораблей (при наличии порта) | = уровень города |
| Вклад в ОД | floor(Σ уровней интегрированных городов / 5) |
| CityPower | = сумма уровней всех интегрированных городов (для экономической победы) |

## Производство юнитов
- Каждый интегрированный город может произвести не более 1 юнита за ход
- Подробные правила: см. `docs/07_units/Production_Rules.md`

## Выбор награды при повышении уровня (v4.13)

При повышении уровня города игрок выбирает **ровно одну** из доступных опций:

### 1) Укрепления (защита города)
- Игрок может купить уровень защиты города (если ещё не достигнут максимум защиты) по ценам из `docs/04_economy/City_Defense.md`.

### 2) Расширение территории города
- `TerritoryRadius += 1`.
- Начальный и минимальный `TerritoryRadius = 1`.
- Правила территории: `docs/03_map/Territory_Rules.md` (v4.13).

### Ограничение Expand Territory (v4.16)

- Награда **Expand Territory (+1 TerritoryRadius)** доступна только если `TerritoryRadius < MaxTerritoryRadius`.
- `MaxTerritoryRadius = 5` (см. `docs/03_map/Territory_Rules.md`).

### 3) Портовая инфраструктура (если есть вода в территории города)
- Не строит порт автоматически.
- Разблокирует/включает возможность построить порт в пределах территории этого города **в обычную фазу действий** (с обычной стоимостью порта и правилами размещения).
- Если в территории города нет воды — опция недоступна.

### Требования к UI
- При апгрейде города отображается выбор одной из опций.
- Если выбранное улучшение недоступно (например, «Порт», но воды нет) — опция должна быть задизейблена.
