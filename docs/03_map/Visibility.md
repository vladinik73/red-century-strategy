# Туман войны и видимость

## Старт
- Игрок видит только область вокруг стартового города/юнита.

## Видимость
- Базовая видимость стартового юнита: 1 клетка вокруг (квадратные тайлы)
- Горы дают +1 видимость юниту, стоящему на горе

> Примечание: точные параметры видимости остальных юнитов задаются в таблице юнитов (по умолчанию — 1).
