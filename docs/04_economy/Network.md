# Магистраль (транспортная сеть)

## Основные правила
- Дороги строятся по клетке (мост = дорога)
- Дороги имеют уровни
- Учитывается **только сеть, связанная со столицей**
- В цивилизации может быть только одна активная сеть (капитальная)

## Пересчёт сети
Пересчёт происходит немедленно при:
- строительстве дороги
- разрушении дороги
- захвате города

Если узел теряет связь со столицей — исключается из бонуса.

## Формула бонуса магистрали
NetworkBonus% =
(ConnectedCities * 2)
+ (TotalRoadTiles * 0.1)
+ (RoadLevelSum * 1)

Ограничение: максимум 40%

## Применение бонуса
- NetworkBonus% влияет на экономику (внутренняя формула доходов)
- NetworkModifier = floor(NetworkBonus% / 10) добавляется к ОД
