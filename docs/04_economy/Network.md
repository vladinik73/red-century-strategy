# Магистраль (транспортная сеть)

## Основные правила
- Дороги строятся по клетке (мост = дорога)
- Дороги имеют уровни
- Учитывается **только сеть, связанная со столицей**
- В цивилизации может быть только одна активная сеть (капитальная)

## Пересчёт сети
Пересчёт происходит немедленно при:
- строительстве дороги
- повреждении дороги (DamagedRoad) или её восстановлении
- захвате города

Если узел теряет связь со столицей — исключается из бонуса.

## Формула бонуса магистрали
NetworkBonus% =
(ConnectedCities * 2)
+ (TotalRoadTiles * 0.1)
+ (RoadLevelSum * 1)

Ограничение: максимум 40%

## Применение бонуса
- NetworkBonus% влияет на экономику (внутренняя формула доходов)
- NetworkModifier = floor(NetworkBonus% / 10) добавляется к ОД

### Стоимость инфраструктуры (канон)
См. канон: `docs/04_economy/Infrastructure_Costs.md`

- Road / Bridge / Port: стоимость в ОД + деньги
- Bridge = ×2 стоимости дороги (и строится только на 1 WATER тайле между сушами)

## Повреждённые дороги (DamagedRoad)

`DamagedRoad` — временный статус инфраструктуры (обычно от кибер-диверсии).

Эффект:
- временно исключается из расчёта `NetworkBonus`
- не даёт бонус перемещения
- автоматически восстанавливается через 2 хода
- может быть восстановлена владельцем за 1 ОД
