# Стабильность и мораль

## Стабильность
- Диапазон: 0–100
- Старт: 70
- Авто-восстановление: +1/ход
### Boost Stability (действие цивилизации, v4.21)

Игрок может ускорить рост стабильности, потратив деньги и 1 ОД.

**Стоимость и эффект:**
- Вариант A: Потратить **50 Money** → `Stability += 2`
- Вариант B: Потратить **100 Money** → `Stability += 5`
- Оба варианта дополнительно требуют: **+1 ОД**
- Ограничение: **не более 1 раза за ход на цивилизацию** (`StabilityBoostUsedThisTurn`)

**Ограничения:**
- Доступно только если у цивилизации есть **хотя бы один интегрированный город** (в т.ч. столица, если она интегрирована).
- Нельзя применить, если денег недостаточно или если ОД < 1.
- `Stability` всегда ограничивается диапазоном **0–100**.

**Тайминг:**
- Выполняется в PHASE 3 (Action Phase), см. `docs/01_overview/Turn_Pipeline.md`.

### Штрафы
- Захват города: -2
- Потеря города: -4
- Потеря столицы: -8
- Разрыв союза: -2
- Каждый оккупированный (неинтегрированный) город: -1

### MoneyMultiplier (влияние на доход денег)

| Стабильность | MoneyMultiplier |
|-------------|-----------------|
| 80–100      | +5%             |
| 60–79       | 0%              |
| 40–59       | -5%             |
| 20–39       | -10%            |
| 0–19        | -20%            |

Используется в формуле: `StabilityMultiplier = 1 + MoneyMultiplier / 100` (см. `docs/04_economy/Money_Model.md`).

## Мораль (связь со стабильностью)
- При стабильности ≥70: мораль = 100
- Каждые 10 пунктов ниже 70: -2 к морали

### MoraleMultiplier (влияние на урон)
Мораль применяется как **множитель** к урону (см. `docs/06_combat/Damage_and_Rules.md`):

| Стабильность | MoraleMultiplier |
|-------------|------------------|
| 80–100      | 1.10 (+10%)      |
| 40–79       | 1.00             |
| 20–39       | 0.90 (-10%)      |
| 0–19        | 0.80 (-20%)      |

## Бунт при нулевой стабильности
- При 0 стабильности: 10% шанс/ход, что город **станет нейтральным**
- Уровень города сохраняется
- Гарнизона нет
- **Defense level сбрасывается в 0** (MVP правило простоты)
- Город нужно повторно захватить
