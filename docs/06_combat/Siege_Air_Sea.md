# Осада, авиация, флот

## Осада
- Если вражеский юнит соседствует с городом → -30% к производству города
- Осада не блокирует развитие полностью

## Авиация
- Базируется только в городе
- Количество авиации в городе = floor(уровень города / 2)
- Может атаковать наземные и морские цели
- Не занимает клетку вне города
- Может быть атакована дальнобойными юнитами (отдельного класса ПВО нет)

## Флот и порты
- Строить корабли можно только в городе с портом
- Лимит кораблей на порт = уровень города
- При захвате города порт сохраняется

### Морской бой и высадка
- Наземные юниты не атакуют корабли
- Корабли могут атаковать города с моря
- Высадка требует порт
- Если порт разрушен: морские юниты не могут базироваться (строительство/базирование запрещено)
