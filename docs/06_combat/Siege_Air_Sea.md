# Осада, авиация, флот

## Осада
- Если вражеский юнит соседствует с городом → город считается «под осадой»
- Полное описание эффектов осады: см. `docs/06_combat/Siege_Effects.md`
- Осада не блокирует развитие полностью

## Авиация
- Базируется только в городе
- Количество авиации в городе = floor(уровень города / 2)
- Может атаковать наземные и морские цели
- Не занимает клетку вне города
- Может быть атакована дальнобойными юнитами (отдельного класса ПВО нет)

## Флот и порты
- Строить корабли можно только в городе с портом
- Лимит кораблей на порт = уровень города
- При захвате города порт сохраняется

### Морской бой и высадка
- Корабли могут атаковать города с моря
- Высадка требует порт
- Если порт разрушен: морские юниты не могут базироваться (строительство/базирование запрещено)

## Взаимодействие суши и флота (канон)

### Атака кораблей с суши
- Наземные юниты ближнего боя **не** атакуют корабли.
- Наземные юниты дальнего боя (**Range ≥ 2**) могут атаковать корабли, если:
  - цель-корабль находится на воде и **примыкает к суше** (coastal tile), и
  - атакующий имеет **видимость** цели.

Примечание:
- Это правило нужно для контр-игры против флота, даже если у стороны нет порта (порт может быть потерян).
