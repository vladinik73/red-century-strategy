# Ветеранство и серийные атаки

## Ветеранство
- Базовый HP юнита задаётся в таблице
- Если юнит убивает 3+ противников за жизнь → повышается максимальный HP (tier-up)

## Серийный удар (Спецподразделение)
- Если цель убита → можно атаковать следующую цель на соседней клетке
- Цепочка продолжается, пока цели умирают и есть соседние цели
- Если цель выжила → цепочка заканчивается
