# Продвинутые юниты (Near-Future)

**Unlock:** Military L4 (Elite Warfare) — все продвинутые юниты (включая Гиперзвук) требуют изучения этой технологии.

## Гиперзвуковая ракета

Параметры: **HP 1 / Damage 7 / Range 4 / Move 0 / Cost 6 ОД / Sight 2**

Правила применения:
- 1 запуск за ход на цивилизацию.
- Требуется **видимость** клетки цели (через юнит/город/горную видимость).
- Целью может быть **только**: наземный юнит или город.
- Нельзя атаковать корабли (флот) гиперзвуком.
- Гиперзвуковая ракета не может быть применена по столице цивилизации.

Правила урона:
- Гиперзвук **игнорирует бонусы местности** (лес/горы) для защиты цели.
- Гиперзвук **не игнорирует городскую оборону** (City Defense Level), если цель находится в укреплённом городе.
- Применяется общий минимум урона (min 1) и общая мораль (MoraleMultiplier).

## Киберподразделение

Параметры: **HP 8 / Damage 0 / Range 1 / Move 2 / Cost 4 ОД / Sight 1**

Особенности:
- Не наносит обычный урон
- Имеет способность **«Сбой»** (см. `docs/07_units/Cyber_Effects.md`)
- Применение «Сбоя» требует **1 ОД**
- Может повредить **1** дорожный узел за ход (`DamagedRoad`)
- Применять можно только на территории врага; применение к нейтральной цивилизации = объявление войны
- «Сбой» **нельзя** применять к столице
- Нельзя повредить дороги в радиусе 1 клетки от столицы
- Нельзя повредить порт, примыкающий к столице
