# Продвинутые юниты (Near-Future)

10) Гиперзвуковой ракетный комплекс (одноразовый)  
6 / 7 / 4 / 1 / 6  
Особенности:
- после атаки уничтожается
- игнорирует бонусы защиты
- НЕ может атаковать столицу
- не более 1 гиперзвукового удара за ход на цивилизацию

11) Рой автономных дронов  
9 / 3 / 2 / 3 / 5  
Особенности:
- игнорирует штраф окружения
- +1 к урону, если рядом есть 2+ роя (бонус ограничен +1 максимум)

12) Киберподразделение  
8 / 0 / 1 / 2 / 4  
Особенности:
- разрушает дорогу/узел магистрали
- накладывает «Сбой» на город: -20% доход, -1 ОД на 2 хода
- не наносит обычный урон
- способность 1 раз за ход, доп. ОД не требует
- применять можно только на территории врага (иначе → объявление войны)
- «Сбой» не применяется к столице
- нельзя разрушать дороги в радиусе 1 клетки от столицы
- нельзя разрушать порт столицы
- повторный «Сбой» по тому же городу: cooldown 3 хода
