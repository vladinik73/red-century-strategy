# Уникальные юниты цивилизаций

Каждая цивилизация имеет 2 уникальных юнита.

Формат: HP / Урон / Дальность атаки / Дальность хода / ОД

## 1) Союз Евразии
- Штурмовая дивизия: 12 / 4 / 1 / 2 / 3  
  Особенность: +1 мораль союзникам в соседних клетках
- Оперативный бронекорпус: 17 / 5 / 1 / 1 / 5  
  Особенность: игнорирует штраф окружения

## 2) Федерация Америк
- Тактический ударный дрон: 8 / 4 / 3 / 2 / 4  
  Особенность: не получает ответного удара при атаке с максимальной дальности
- Экспедиционный мобильный корпус: 11 / 4 / 1 / 3 / 4  
  Особенность: +1 урон вне своей территории

## 3) Южноамериканский Союз
- Амазонская ударная группа: 9 / 3 / 2 / 2 / 3  
  Особенность: +2 защита в лесу
- Континентальный оборонительный модуль: 16 / 4 / 1 / 1 / 4  
  Особенность: -1 входящий урон на своей территории

## 4) Африканская Коалиция
- Мобильная тактическая колонна: 9 / 3 / 1 / 4 / 3  
  Особенность: игнорирует штраф движения по пустыне
- Пустынный автономный рейдер: 8 / 3 / 1 / 3 / 3  
  Особенность: после атаки может отступить на 1 клетку

## 5) Европейская Федерация
- Инженерно-логистический корпус: 10 / 2 / 1 / 2 / 3  
  Особенность: строительство дорог на 50% дешевле
- Высокоточная ракетная батарея: 8 / 5 / 3 / 1 / 4  
  Особенность: +1 урон по городам

## 6) Южноазиатский Союз

### Южноазиатский Союз — Массовый мобилизационный батальон (v4.6)

Параметры:
- HP: 7
- Damage: 2
- Range: 1
- Move: 2
- Cost: 1 ОД

Особенности:
- Является самым дешёвым юнитом в игре.
- Не предоставляет бонуса к лимиту юнитов в городе (удалено в v4.6).
- Ударная волна прорыва: 11 / 4 / 1 / 2 / 3  
  Особенность: при гибели снижает мораль врага на 5

## 7) Китайская Гармония
- Страж внутренней гармонии: 14 / 3 / 1 / 1 / 3
  Особенность: +2 к защите города

### Китайская Народная Гармония — Нефритовый автономный страж (v4.7)

Параметры: (без изменений)
- HP: 18
- Damage: 5
- Range: 1
- Move: 1
- Cost: 5 ОД

Особенность (обновлено в v4.7):
- `+1` к стабильности за убийство, **только если** убийство произошло:
  - в интегрированном городе Китайской Народной Гармонии, **или**
  - на территории Китайской Народной Гармонии.
