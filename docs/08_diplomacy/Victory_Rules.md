# Победы

Проверка условий победы и обновление победных таймеров выполняются **в конце каждого хода цивилизации**, а не в конце полного раунда.

См. канон:
- `docs/01_overview/Turn_Pipeline.md` → PHASE 4 (End of Turn)

При одновременном выполнении нескольких побед приоритет:
**Военная > Экономическая > Технологическая**.

## Victory type (v4.41)

`victory_type` — **производное значение** из событий, не хранится в `match.victory.per_player`.

- **Источник:** `VICTORY_COMPLETE` event (payload: `victory_type`, `player_id`).
- **Значения в event:** `MILITARY` | `ECONOMIC` | `TECH`.
- **Значения для UI/display:** `MILITARY` | `ECONOMIC` | `TECHNOLOGICAL` (TECH) | `ALLIANCE`.
- **ALLIANCE:** когда побеждает лидер альянса; тип отображения = ALLIANCE.

## Военная победа
- PvE: контроль ≥60% интегрированных городов

## Экономическая победа (канон)

Условие:
- Под контролем игрока должно быть **≥ 75% глобальной городской мощи**.

Определения:
- `CityPower = Σ уровней интегрированных городов` (считаются только интегрированные города).
- `GlobalCityPower = Σ CityPower всех цивилизаций` (по интегрированным городам).

Правило подсчёта:
- В расчёт **НЕ** входят города союзников.
- Союз влияет на экономику (доход/мораль/ОД), но **не считается контролем** для условия победы.

Таймер:
- После достижения порога запускается таймер **5 ходов удержания**.
- Если в любой момент значение падает ниже 75%, таймер сбрасывается.

## Технологическая победа
- После изучения 5 уровня научной ветки запускается таймер 10 ходов
- Любой захват **интегрированного** города этой цивилизации сбрасывает таймер
- Если город возвращён прежнему владельцу в 4 хода интеграции — таймер не сбрасывается

## Победа альянса
- Если условие выполнено альянсом — победителем считается лидер альянса
- Если лидер альянса — ИИ, игрок проигрывает
