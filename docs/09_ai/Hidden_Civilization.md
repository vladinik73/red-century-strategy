# Скрытая цивилизация (AI-only)

## Правила появления
- Может появиться после 20-го хода
- Появляется в 1–3 регионах
- Получает 2–3 нейтральных города
- Повышенная агрессия
- Может победить

## Назначение в дизайне
- Создаёт mid/late-game угрозу, ломает «устоявшиеся» фронты
- Наказывает слишком жадный макро-рост без армии
