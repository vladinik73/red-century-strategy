# UI/UX — экран города (черновик структуры)

> Этот раздел задаёт каркас UI города. Детальная UX-проработка делается итеративно.

Показывать всегда:
- Название города, уровень
- Защита
- Интеграция (если есть: оставшиеся ходы)
- Порт (есть/нет)
- Лимит юнитов (уровень города) и текущее заполнение
- Доступные действия (зависят от ОД/денег/интеграции)

Кнопки:
- Произвести юнит (если город интегрирован и есть ОД)
- Улучшить город (если доступны ресурсы/деньги)
- Защита города (уровни)
- Порт (строительство)
- Специализация (выбор на повышении уровня)

Подсветка территории города на карте при открытой панели.
