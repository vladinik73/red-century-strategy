# Keyboard Shortcuts (v4.40)

Канон: `docs/10_uiux/Main_Game_Screen.md`. Конфликты и политика переназначения.

---

## 1) Keymap table

| Shortcut | Action | Примечание |
|----------|--------|------------|
| Enter / Space | End Turn | Основное действие |
| N / Tab | Cycle next unit | Следующий юнит (не acted) |
| P / Shift+Tab | Cycle next city | Следующий город |
| 1–4 | Open overlay | Network, Resources, Threat, Integration (опционально) |
| 0 | Clear overlay | Выключить overlay |
| + / = | Zoom in | |
| - | Zoom out | |
| Esc | Deselect / Close panel | Снять выделение, закрыть модал |

---

## 2) Conflicts

- Esc: закрывает модал, затем снимает выделение (приоритет: модал).
- Enter: в текстовом поле — новая строка; вне поля — End Turn.
