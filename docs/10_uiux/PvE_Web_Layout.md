# UI/UX PvE Web — Layout

## Стиль
- Минималистичный (Polytopia-inspired)
- Светлая тема
- Реалистичная, но упрощённая карта (без мелких деталей)
- Нет границ стран: только границы городов

## Основной экран
- Верхняя панель: ресурсы, ОД, стабильность, ход
- Левая панель «Империя»: кол-во городов, CityPower, NetworkBonus%, кол-во юнитов (сворачиваемая)
- Центральная карта
- Правая контекстная панель (сворачиваемая; авто-раскрывается при выборе объекта)
- Нижняя панель: лог / технологии / дипломатия

## Оверлеи карты
Переключатели:
- Сеть
- Ресурсы
- Военная угроза
- Интеграция

## Территории
- Территория принадлежит городу
- Граница города — тонкий цветной контур
- При выборе города контур усиливается, территория подсвечивается
