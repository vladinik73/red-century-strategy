# Settings Screen (v4.40)

Канон: `docs/01_overview/MVP_Player_Journey.md` (Home → Settings).

---

## 1) Entry point

- Кнопка **Settings** на экране Home.
- Открывает экран настроек (модал или отдельная страница).

---

## 2) Settings list (minimal web MVP)

| Настройка | Тип | Описание |
|-----------|-----|----------|
| Audio: Music | Toggle | Вкл/выкл музыка |
| Audio: SFX | Toggle | Вкл/выкл звуки |
| Graphics | Low / High | Уровень детализации (placeholder) |
| Keybinds | Link | Открыть `docs/10_uiux/Keyboard_Shortcuts.md` или экран переназначения |
| Accessibility: Reduced motion | Toggle | Уменьшить анимации |
| Accessibility: High contrast | Toggle | Высокий контраст UI |

---

## 3) Confirm / Close

- **Apply** — применить и закрыть.
- **Cancel** — отменить изменения и закрыть.
- **Close** — то же, что Cancel, если нет несохранённых изменений.
