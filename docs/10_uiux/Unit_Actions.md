# UI/UX — действия юнита

**v4.40:** UI pack: Unit_Visual_Spec, Combat_Feedback_UI.

## Unit Actions: Heal & Disband (v4.20)

Общее правило (MVP):
- У каждого юнита **1 действие за свой ход**: юнит либо перемещается, либо атакует, либо лечится, либо расформировывается, либо выполняет спец-действие (если есть).
- После выполнения действия юнит помечается как `HasActedThisTurn = true` и больше ничего не делает до следующего своего хода.

### Heal (Вылечить)
- Доступно, если `CurrentHP < MaxHP`.
- Стоимость: **1 ОД**.
- Эффект: `CurrentHP = min(MaxHP, CurrentHP + 3)`.
- Требует действие юнита (после лечения юнит больше не может двигаться/атаковать в этот ход).

### Disband (Расформировать)
- Стоимость: **0 ОД** (но требует действие юнита).
- Эффект: юнит удаляется с карты.
- Возврат: игрок получает назад **50% AP Cost** юнита:
  `RefundAP = floor(UnitAPCost × 0.5)`.
- Возврат добавляется в **текущий** пул доступных ОД цивилизации в этот ход.
- Disband нельзя отменить (no-undo).
