# UI/UX — взаимодействие с юнитами

**v4.40:** UI pack: Unit_Visual_Spec, Combat_Feedback_UI.

## Выбор юнита
- Подсвечиваются **только клетки движения**
- Перемещение мгновенное (без анимации)
- Отмена перемещения/атаки невозможна

## Атака
- После перемещения часть юнитов может атаковать соседние клетки (если враг в радиусе)
- Атака выполняется кликом по врагу при выбранном юните
- Нет кнопок «Атаковать» / «Переместить»

## Панель юнита (правая)
- Название
- HP (только число)
  - если HP ≤ 30% max → число красное
- Кнопки:
  - Вылечить
  - Расформировать

## Серийный юнит (Serial Strike, канон v4.4)
- После убийства цели атакующий занимает клетку убитого юнита и может атаковать следующую соседнюю цель
- Максимум убийств за цепочку: `SerialKillCap = 5`
- После каждого убийства урон следующей атаки снижается: `DamageMultiplierPerKill = 0.85` (-15%)
- Цепочка прекращается, если: цель выжила, нет соседних врагов или достигнут лимит 5 убийств
- Если последний защитник города убит серийным ударом, появляется выбор:
  - **Capture Now** (цепочка завершается, город захвачен),
  - **Continue Chain** (захват откладывается до завершения цепочки; захват происходит только если юнит остался на клетке города).
