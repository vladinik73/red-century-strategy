# RED AGE
## Production System Specification v2.0

Документ предназначен для прямой передачи в разработку (Claude / Cursor / Engine Team).
Содержит финальную архитектуру механик, формулы, ограничения и поведение ИИ.

============================================================
I. РЕЖИМЫ И ПАРАМЕТРЫ КАРТЫ
============================================================

PvE:
- 80x80
- 4–6 континентов
- 5–10 островов
- 50–100 городов
- ≥10 клеток по оси между столицами

PvP:
- 50x50
- 3–4 континента
- 3–5 островов
- 30–50 городов
- ≥7 клеток по оси между столицами

Города:
- Предгенерация
- ≥2 клетки между городами
- Нельзя строить вручную
- Только захват

============================================================
II. ГЛОБАЛЬНЫЕ РЕСУРСЫ
============================================================

1. Деньги (накопительные)
2. Наука (прирост за ход)
3. Очки Действия (ОД)
4. Стабильность (0–100)
5. Мораль

Формула ОД за ход:
ОД = База (5)
+ floor(Сумма уровней интегрированных городов / 5)
+ floor(Количество технологий / 3)
+ Бонус стабильности
+ Бонус магистрали

Где:
- Стабильность 80–100: +1 ОД
- 20–39: -1 ОД
- 0–19: -2 ОД

============================================================
III. ГОРОДА
============================================================

Уровень города:
- Лимит юнитов в городе = уровень
- Производство: 1 юнит за ход
- Если город свободен (нет гарнизона) — производство мгновенное

При повышении уровня выбрать:
1. Расширение территории
2. Защитное укрепление (+2 к защите города)
3. Специализация (экономика +10% доход города)

============================================================
IV. ИНТЕГРАЦИЯ
============================================================

После захвата:
- 4 хода интеграции
- Нет производства
- Нет магистрали
- Не учитывается в победе

Если возвращён прежнему владельцу в 4 хода — штрафы отменяются.

============================================================
V. МАГИСТРАЛЬ
============================================================

Учитывается только сеть, связанная со столицей.
Существует только одна активная сеть.

Формула бонуса магистрали:
NetworkBonus% =
(ConnectedCities * 2)
+ (TotalRoadTiles * 0.1)
+ (RoadLevelSum * 1)

Ограничение: максимум 40%

Disconnected city → теряет бонус.

Мост = дорога.

============================================================
VI. БОЙ
============================================================

HP базовый: 10

Урон базовый: 3

Мораль влияет на урон:
- 80–100: +10%
- 20–39: -10%
- 0–19: -20%

Окружение:
- 4+ врагов из 8 соседних клеток → -1 к защите

Горы:
- +1 обзор
- +1 урон при атаке с горы
- -1 урон при атаке на гору с равнины
- Вход возможен после Военной технологии уровня 2

Серийный удар:
- Если цель убита → возможна следующая атака
- Неограниченная цепочка, пока цель умирает

Осада:
- Если вражеский юнит соседствует с городом → -30% к производству

============================================================
VII. ТЕХНОЛОГИИ
============================================================

Военная:
1. Улучшение пехоты
2. Доступ к горам
3. Осадные технологии
4. Элитные войска
5. Военная доктрина

Экономическая:
1. Дороги I
2. Порты
3. Дороги II
4. Магистральная логистика
5. Инфраструктурный максимум

Научная:

2. +10% науки
3. +20% науки
4. Ускорение технологий
5. Технологический прорыв

Тех-победа:
- После уровня 5
- Таймер 10 ходов
- Каждый захват города этой цивилизации сбрасывает таймер

XV. ПОЛНАЯ МАТЕМАТИЧЕСКАЯ МОДЕЛЬ ИИ
============================================================

LeaderIndex =
0.4 * Military
+ 0.35 * Economy
+ 0.25 * Progress

Military =
0.6 * (Доля интегрированных городов)
+ 0.4 * (Сила армии)

Economy =
0.7 * (Доля суммарных уровней городов)
+ 0.3 * (NetworkBonus%)

Progress =
0.5 * (Уровень научной ветки / 5 * 100)
+ 0.3 * (Общий уровень технологий / 15 * 100)
+ 0.2 * (Прогресс тех-таймера)

ThreatScore = LeaderIndex + ProximityBonus

EnemyScore =
0.45 * ThreatScore
+ 0.25 * ProximityScore
+ 0.20 * OpportunityScore
+ 0.10 * GrievanceScore

Пороги атаки:
Easy: 70
Normal: 55
Hard: 45
God: 35 (30 при стратегической цели)

CityPriority =
30 * CapitalFlag
+ 20 * NetworkNodeValue
+ 20 * VictoryDenialValue
+ 15 * WeakDefenseValue
+ 15 * StrategicPositionValue

WarScore =
0.55 * OpportunityScore
+ 0.25 * max(0, ThreatScore-50)
+ 0.20 * Revenge

AllyScore =
0.45 * BenefitScore
+ 0.25 * RelationScore
+ 0.30 * AntiLeaderNeed

BreakScore =
0.5 * BetrayalGain
+ 0.3 * ThreatScore
+ 0.2 * OpportunityElsewhere

============================================================
XXI. ФОРМУЛА БОЯ (ПОДРОБНО)
============================================================

Итоговый урон рассчитывается:

Damage = BaseDamage
+ TerrainBonus
+ MoraleModifier
- DefenseModifier

Где:
BaseDamage = параметр юнита
TerrainBonus:
+1 при атаке с горы
-1 при атаке на гору с равнины
MoraleModifier:
80–100 стабильности → +10%
20–39 → -10%
0–19 → -20%
DefenseModifier:
+1 если юнит в лесу
+2 если в укреплённом городе
-1 если окружён 4+ врагами

Минимальный урон = 1
Округление вниз.

============================================================
XXII. АВИАЦИЯ
============================================================

- Базируется только в городе
- Количество авиации в городе = floor(уровень города / 2)
- Может атаковать наземные и морские цели
- Не может занимать клетки вне города
- Может быть атакована дальнобойными юнитами

============================================================
XXIII. МОРСКОЙ БОЙ И ВЫСАДКА
============================================================

- Наземные юниты не атакуют корабли
- Корабли могут атаковать города с моря
- Высадка требует порт
- При разрушении порта морские юниты не могут базироваться

============================================================
XXIV. КИБЕРПОДРАЗДЕЛЕНИЕ (ДЕТАЛИ)
============================================================

- Использование способности 1 раз за ход
- Действие не требует дополнительных ОД
- Можно применять только на территории врага
- "Сбой" действует 2 хода
- Повторное наложение возможно после окончания эффекта
- Не действует на столицу

============================================================
XXV. ДОБЫЧА РЕСУРСОВ С КЛЕТОК
============================================================

- Каждая ресурсная клетка имеет запас (10 единиц)
- Добыча: 2 единицы за ход
- После истощения → low productivity (1 единица за ход)
- Запуск добычи требует 1 ОД
- Дальнейшая добыча ОД не требует

============================================================
XXVI. УНИЧТОЖЕНИЕ ЦИВИЛИЗАЦИИ
============================================================

При захвате последнего города:
- Все юниты исчезают
- В системный лог выводится сообщение
- Отображается статистика:
  - Максимальное число городов
  - Пик мощности
  - Количество ходов существования
  - Лидер цивилизации

============================================================
XXXI. ОЧКИ ДЕЙСТВИЯ (ПОЛНАЯ МОДЕЛЬ)
============================================================

ОД за ход рассчитываются:

OD = 5 (база)
+ floor(Сумма уровней интегрированных городов / 5)
+ floor(Количество технологий / 3)
+ StabilityModifier
+ NetworkModifier
+ AllianceModifier
- OccupationPenalty
- CyberPenalty

Где:
StabilityModifier:
80–100 → +1
20–39 → -1
0–19 → -2

NetworkModifier = floor(NetworkBonus% / 10)
AllianceModifier = +1 если активен союз
OccupationPenalty = -1 за каждые 2 неинтегрированных города
CyberPenalty = -1 при активном «Сбое»

============================================================
XXXII. ЭКОНОМИЧЕСКАЯ ПОБЕДА (ФОРМАЛИЗАЦИЯ)
============================================================

CityPower = сумма уровней интегрированных городов

Победа при контроле (прямо или через альянс)
>= 75% суммарного CityPower карты

Таймер удержания: 5 ходов подряд
Проверка в конце хода
При падении ниже 75% таймер сбрасывается

============================================================
XXXIII. ТЕХНОЛОГИЧЕСКАЯ ПОБЕДА (УТОЧНЕНИЕ)
============================================================

После изучения 5 уровня научной ветки запускается таймер 10 ходов.
Любой захват интегрированного города этой цивилизации
сбрасывает таймер.

Если город возвращён в течение 4 ходов интеграции —
таймер не сбрасывается.

============================================================
XXXIV. МАГИСТРАЛЬ — АЛГОРИТМ ПЕРЕСЧЁТА
============================================================

Пересчёт сети происходит при:
- строительстве дороги
- разрушении дороги
- захвате города

Пересчёт происходит немедленно.

Если узел теряет связь со столицей —
он исключается из NetworkBonus.

============================================================
XXXV. ДИВЕРСИИ И ОБЪЯВЛЕНИЕ ВОЙНЫ
============================================================

Использование киберподразделения на территории нейтральной
цивилизации автоматически объявляет войну.

============================================================
XXXVI. ЗАЩИТА ГОРОДА (УРОВНИ)
============================================================

Уровни защиты:
1 уровень: +2 защита (стоимость 100 денег)
2 уровень: +3 защита (стоимость 200 денег)

Осада снижает бонус защиты на 1.

============================================================
XXXVII. ОКРУЖЕНИЕ (УТОЧНЕНИЕ)
============================================================

Учитываются 8 соседних клеток (включая диагонали).
При 4+ врагах → -1 к защите.
Если один из окружения уничтожен — штраф снимается.

============================================================
XXXVIII. ФЛОТ И ПОРТЫ
============================================================

- Строительство кораблей возможно только в городе с портом.
- Лимит кораблей на порт = уровень города.
- При захвате города порт сохраняется.

============================================================
XXXIX. УСЛОВИЕ УНИЧТОЖЕНИЯ
============================================================

Если уничтожена последняя интегрированная столица —
цивилизация уничтожается.
Юниты исчезают.

============================================================
XL. ПОБЕДА АЛЬЯНСА
============================================================

Если победное условие выполнено альянсом —
победителем считается лидер альянса (игрок).
Если лидер ИИ — игрок проигрывает.

============================================================
XLI. ПРОВЕРКА ПОБЕДЫ
============================================================

Проверка всех условий производится в конце полного раунда.
Если несколько условий выполняются одновременно —
приоритет: Военная > Экономическая > Технологическая.

============================================================
XLII. ГЕНЕРАЦИЯ КАРТЫ
============================================================

- Всегда 50–100 городов (PvE)
- Всегда 30–50 городов (PvP)
- Минимум 4 континента
- Минимум 5 островов
- Алгоритм стремится обеспечить первый контакт до 20 хода.

============================================================
XLIII. КИБЕРПОДРАЗДЕЛЕНИЕ (COOLDOWN)
============================================================

После применения «Сбоя» повторное использование
на тот же город возможно через 3 хода.

============================================================

============================================================
XLIV. ДОПОЛНИТЕЛЬНЫЕ УТОЧНЕНИЯ БАЛАНСА
============================================================

1. КИБЕРПОДРАЗДЕЛЕНИЕ — ОГРАНИЧЕНИЯ
- Нельзя применять «Сбой» на столицу.
- Нельзя разрушать дороги в радиусе 1 клетки от столицы.
- Нельзя разрушать порт столицы.

2. АВИАЦИЯ
- Дополнительные классы ПВО не вводятся.
- Перехват осуществляется стандартными дальнобойными юнитами.

3. РОЙ АВТОНОМНЫХ ДРОНОВ
- Бонус к урону от группировки ограничен +1 максимум.
- Независимо от количества соседних роя.

4. ГИПЕРЗВУКОВОЙ РАКЕТНЫЙ КОМПЛЕКС
- Не может наносить удар по столице.
- За ход может быть применён только один гиперзвуковой удар на цивилизацию.

5. ВОССТАНИЕ ГОРОДА
- При 0 стабильности и успешной проверке (10%) город становится нейтральным.
- Уровень города сохраняется.
- Гарнизон отсутствует.
- Требуется повторный захват.

============================================================

============================================================
XLV. UI/UX PvE WEB (MINIMALISTIC MODE)
============================================================

Общий стиль:
- Минималистичный интерфейс (вдохновлён Polytopia)
- Светлая цветовая схема
- Реалистичная, но упрощённая карта
- Отсутствуют границы стран (есть только границы городов)

LAYOUT:
- Верхняя панель ресурсов (фиксированная)
- Левая панель "Империя" (скрываемая)
- Центральная карта
- Правая контекстная панель (скрываемая)
- Нижняя панель (лог, технологии, дипломатия)

БОКОВЫЕ ПАНЕЛИ:
- Левая и правая панели могут сворачиваться
- При выборе объекта правая панель автоматически раскрывается

КАРТА:
- Плоские цвета без мелких текстур
- Территории принадлежат городам
- Границы города отображаются тонким цветным контуром
- Overlay режимы: Сеть / Ресурсы / Военная угроза / Интеграция

ПОВЕДЕНИЕ ЮНИТА:
- При выборе юнита подсвечиваются только клетки движения
- Перемещение мгновенное (без анимации)
- Отмена перемещения невозможна
- Если после перемещения враг в радиусе атаки — юнит остаётся активным
- Атака производится кликом по врагу
- Нет кнопок "Атаковать" или "Переместить"

ПАНЕЛЬ ЮНИТА:
Отображает:
- Название
- Текущий HP (только цифра)
- Кнопки: Вылечить, Расформировать

HP отображение:
- Только числовое значение
- Если HP <= 30% от максимума — цифра красная

СЕРИЙНЫЙ ЮНИТ:
- После убийства цели остаётся активным
- Может атаковать следующую цель
- Цепочка продолжается до момента, когда цель не погибает

КНОПКА "ВЫЛЕЧИТЬ":
- Стоимость: 1 ОД
- Восстанавливает +3 HP
- Нельзя использовать, если рядом враг

КНОПКА "РАСФОРМИРОВАТЬ":
- Юнит уничтожается
- Возвращает 50% стоимости в деньги

АТАКА:
- Разрешается мгновенно
- Отображаются всплывающие числа урона
- Отмена невозможна

ЗАВЕРШЕНИЕ ХОДА ЮНИТА:
- После атаки юнит становится неактивным
- Активный юнит выделяется тонким светлым контуром
- После завершения хода контур исчезает
